2023

As a Unity game developer at Plundervolt from May 2023 to December 2023, I conducted systems analysis and gameplay improvements. I implemented front-end features in collaboration with artists and developed complex systems such as skill trees, Breadth First Search movement, and battle systems for a Turn-based RPG. Applying optimization practices like SOLID, I ensured gaming performance and scalability. Working in multidisciplinary teams, I resolved technical and creative challenges to deliver the final product.

Graduated in Digital Games, I learned more about designing systems, implementing design patterns, applying AI concepts to games and integrating with external tools, such as API and database.

2020 - 2022

As a software developer at Zoyd Software from September 2022 to January 2023, focusing on enhancing system functionalities and creating game projects. I developed virtual and augmented reality experiences, integrated NFTs via APIs into augmented reality apps, and designed collision systems for isometric combat games. Using APIs, databases, and Firebase Analytics, I optimized projects. I also prioritized documentation, code refactoring, and coding standards.

As a web developer at IDSingular from September 2020 to January 2022, I focused on front-end implementation in collaboration with UX designers to create web interfaces for gamers. I developed and optimized games for web browsers, using JavaScript, Node.js, Socket.IO, and Firebase for API and database connections. I implemented data encryption for security, developed multiplayer systems with evaluators creating rooms, and designed ranking, evaluation, and trial systems. Additionally, I managed file uploads and database integration for exams, actively participating in testing and debugging for game quality assurance.

2020 - 2021

I developed the game "When are you coming?" In a 14-day gamejam, I designed and developed the entire game alone, the idea was to represent a couple separated by the pandemic trying to keep their relationship alive.

I developed the game "Hope" in a 14-day gamejam "Game JAAJ 5", I designed and developed the entire game alone, this game was made at a time when I was depressed, so the idea was to represent the loop of a person who feels that things never get better.

2019 - 2020

As an instructor at Magic Star - School from August 2019 to March 2020, I managed programming and game design courses for children and teenagers. I coordinated workshops, developed gamification for student assessment, and provided individualized support. Collaborating with the pedagogical team, I contributed to educational strategies and simplified the creation of gamification questionnaires for teachers.

2018

I completed a Technical Course in Digital Games Programming at ETECAF - Etec Aristóteles Ferreira in 2018. I specialized as a digital game developer and was recognized as a finalist for best TCCs (Final Course Projects) at the institute.
